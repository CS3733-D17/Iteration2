Create Account Help:

Please fill the specified fields.



First Name Example: Bob
Last Name Example: Wong

Email Address Example: name@email.com
Password: any desired characters (ex: Peter1234!)

Label Search Help:

Search Field: Enter the name of the brand you want to search for


Advanced:
Beverage Type: Select From “All”, “Wine”, “Beer”, or “Distilled”

Select Search Type: “Brand Name” or “Fanciful Name” or “Either”
Select Product Source: “DOMESTIC” or “IMPORTED”
Origin Code: Enter either the numerical code or the description of the origin code. Autocomplete options will be shown.
Product Category Code: Enter either the numerical code or the description of the product category code. Autocomplete options will be shown.

Alcohol Content: Search for alcohol content “equal to” or “between”
Values: Enter the alcohol content value(s) for the search

Wine Only:
pH Level Select a pH level “equal to” or “between”
Values: Enter the pH level(s) for the search

Application Form Help:

Representative ID: Your employee identification number for the manufacturer that you work for
Phone Number: Your work phone number (###-###-####)
Plant Number: Plant Identification Number (@@-@@-####)
Email Address: email where name@email.com

Application Address: (Address of applicant)
Location Name (ex: manufacturer)
Street Address (ex: 23 Hickory Drive)
Town State, Zip (ex: Worcester MA, 01541)
Mailing Address: (if different from Application Address)
Location Name (ex: manufacturer)
Street Address (ex: 23 Hickory Drive)
Town State, Zip (ex: Worcester MA, 01541)

Serial Number: ##-####
Serial Number Example: 55-1234

Alcohol content: 0-100(%)
AC Example: 10.4

State: The state that the company you work for is located in (ex: MA)

Bottle capacity: The capacity of the bottle (mL)
Capacity Example: 250

TTB ID: The identification number assigned to the COLA that you work for (ex: 12345678901234)

Brand Name Example: Samuel Adams

Fanciful Name Example: Pinot Noir

Additional Information: any additional information about the application

Application Edit Help:


Allowed Revisions: select all fields you wish to revise
The same restrictions to fields apply as did when creating the application.
Submit Revisions: submit the edited application for review in place of the old application.

Import an Existing Application:


  • Click the import button at the top of the create application page
  • Enter the existing TTB number into the text field
  • Click import to begin importing the form
  • Fill in any fields that were not found in the existing form with your label information
  • Click “Submit” when you are finished.

Application Processing Help:


Accept: Accept the application and allow the label to be searched
Send for Corrections: Send the application back to the applicant for corrections before accepting it
Reject: Reject the application and prevent it from ever showing up on a search
Send for Second Opinion: If you are unsure, send it to another employee for review with comments
Second Opinion Employee: Email of employee desired to give a second opinion. This will provide autocomplete options

Application Processing Workflow:


An employee can look at the list of applications they have been assigned, and can review them. The name of the label application and some basic information is located in each box.
Click the blue “Review Application” button to review the application.
Employees get applications limited to ten at a time.
New applications will be automatically pulled when all current forms in the inbox are finished.
