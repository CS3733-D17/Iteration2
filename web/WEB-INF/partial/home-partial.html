Create an Account

You can sign up as a manufacurer or US employee in order to gain access to more function.

Submiting a Label

Once you have signed up as a manufacturer, you can then create an application and submit it to be reviewed by a US employee.

Reviewing Applications

As a US employee, you will be able to approve or reject applications made by manufacurers.


I submitted a label application a week ago. Did it not go through?

It may take several weeks to process your application. If you are worried that it did not get through the system, go to our help page.

There were errors in my label application. Can I edit it?

Yes. You can revise your application by clicking the edit button on the application you would like to edit

Is there a way to make sure my label was approved?

You should be able to see if your application was approved or rejected. However, if you are unsure, you can use the COLA search to look up your label.

I'm having trouble submitting an application. What do I do?

You can go to our help page or call 373-337-3337 for more information.
