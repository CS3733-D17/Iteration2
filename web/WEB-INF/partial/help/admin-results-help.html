Admin Results Help

This is the page which contains the information for an approved label. At the top of the page is the label that was submitted for approval. Below that is the following information.


Beverage Type: The identifies the type of the beverage. This will be either "Wine", "Beer", or "Distilled".


Beverage Source: This identifies whether the beverage is manufactured in the United States, "Domestic", or in another country, "Imported".


Cureent Status: This identifies whether or not the approval for the label is still valid or not.


Alcohol Content: This is the percentage of the beverage that is alcohol.


Approval Date: This is the date on which the label was approved.


Expiration Date: This is the date on which the approval will stop being valid.


Approving Agent: This is the name of the agent that approved the label.
